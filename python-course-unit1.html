什么是Python？

Python是一种简单易学、功能强大的编程语言。它被广泛应用于：

  • 数据分析和数据科学
  • 人工智能和机器学习
  • Web开发
  • 自动化脚本
  • 游戏开发

Python的特点：

  • 语法简洁：代码可读性高，容易理解
  • 跨平台：可以在Windows、Mac、Linux上运行
  • 丰富的库：有大量现成的工具可以使用
  • 社区活跃：遇到问题容易找到帮助

步骤1：安装Python 3.11+

1.1 Windows系统

步骤：

  1. 访问 Python官网
  2. 点击"Download Python 3.11.x"按钮（x是具体版本号）
  3. 运行下载的安装程序
  4. 重要：勾选"Add Python 3.11 to PATH"选项
  5. 点击"Install Now"
  6. 安装完成后，点击"Close"
💡 提示

勾选"Add Python to PATH"非常重要，这样你就可以在命令提示符中直接运行Python。

1.2 Mac系统

步骤：

  1. 访问 Python官网
  2. 点击"Download Python 3.11.x"按钮
  3. 运行下载的.pkg安装包
  4. 按照安装向导的提示完成安装
1.3 Linux系统

步骤：

  1. 打开终端
  2. 运行命令：sudo apt update（Ubuntu/Debian）
  3. 运行命令：sudo apt install python3 python3-pip
  4. 验证安装：python3 --version
1.4 验证安装

安装完成后，我们需要验证Python是否正确安装：

  • Windows：打开命令提示符（Win+R → 输入cmd → 回车），输入 python --version
  • Mac/Linux：打开终端，输入 python3 --version

如果看到类似 Python 3.11.x 的输出，说明安装成功！

步骤2：安装VS Code编辑器

VS Code是一个免费、强大的代码编辑器，非常适合Python编程：

  1. 访问 VS Code官网
  2. 下载适合你系统的版本
  3. 运行安装程序并按照提示完成安装
2.1 安装Python扩展

打开VS Code后，我们需要安装Python扩展：

  1. 点击左侧的扩展图标（方块形状）
  2. 在搜索框中输入"Python"
  3. 点击第一个结果（由Microsoft提供）
  4. 点击"Install"按钮

步骤3：编写并运行第一个Python程序

3.1 创建项目文件夹
  1. 在电脑上创建一个名为"python-learning"的文件夹
  2. 在VS Code中，点击"File" → "Open Folder"，选择这个文件夹
3.2 创建Python文件
  1. 在VS Code左侧的文件浏览器中，点击"New File"按钮
  2. 输入文件名：hello.py
  3. 按Enter键确认
3.3 编写代码

在hello.py文件中输入以下代码：

hello.py
# 这是我的第一个Python程序 print("Hello, World!") print("欢迎来到Python世界！")
3.4 运行程序

有两种方法运行Python程序：

方法1：使用VS Code的运行按钮

  1. 在VS Code右上角，点击绿色的运行按钮（播放图标）
  2. 选择"Run Python File in Terminal"

方法2：使用终端

  1. 在VS Code中，点击"Terminal" → "New Terminal"
  2. 在终端中输入：python hello.py（Windows）或 python3 hello.py（Mac/Linux）
  3. 按Enter键

预期输出：

预期输出
Hello, World! 欢迎来到Python世界！

常见问题与解答

❓ 常见问题
Q1: 运行Python时出现"python不是内部或外部命令"怎么办？
A: 这是因为Python没有添加到系统PATH中。重新运行Python安装程序，确保勾选了"Add Python to PATH"选项。
Q2: VS Code找不到Python解释器怎么办？
A: 在VS Code右下角，点击Python版本号，然后选择正确的Python解释器路径。
Q3: 代码运行时出现语法错误怎么办？
A: 仔细检查代码，确保：
  • 括号、引号都是成对的
  • 缩进正确（Python对缩进很严格）
  • 没有拼写错误
Q4: 我可以使用其他编辑器吗？
A: 当然可以！除了VS Code，你还可以使用：
  • PyCharm（专业的Python IDE）
  • Sublime Text
  • Atom
  • 甚至记事本（不推荐）
